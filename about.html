My Story

I started with basic Linux system administration and quickly became fascinated with how attackers think. That curiosity pulled me into CTF wargames — specifically OverTheWire Bandit, which I've completed in full. Every challenge taught me something concrete: file encoding chains, SSH key manipulation, privilege escalation techniques, compression analysis.

Learning security hands-on — not just reading theory — changed how I think about software. I now build every project with a security mindset: what can go wrong, how can this be exploited, and how do I detect it happening in real time. SentinelX came directly from that mindset.

I'm actively looking for internship or junior opportunities in cybersecurity where I can contribute, learn in a professional environment, and grow toward a security engineering role.
